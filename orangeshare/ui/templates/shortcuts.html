Shortcuts

Orange Share uses several shortcuts for different parts of its functionality. These shortcuts have to be installed on the device from which content shall be shared.

To install a shortcut, click on a download button below and scan the QR Code with your device's camera app. Click the popup and follow the instructions on screen.

Untrusted Shortcuts

To import the needed shortcuts, you will have to enable Allow Untrusted Shortcuts in your devices settings. This option can be found in Settings > Shortcuts.

If you have never before used Shortcuts, chances are, that Allow Untrusted Shortcuts is grayed out. If this is the case, you have to run a shortcut first. To do this, open the Shortcuts App and choose any of the available shortcuts or create one yourself. Once you ran a shortcut, the option to enable untrusted shortcuts should become available.

How to use the Shortcuts

Once the shortcuts are installed, they are shown in the Shortcuts App. Here you can start or edit them. Apart from the Connection Manager, starting the shortcuts from the app won't do anything as they need some input to share.

You can find all shortcuts other than the Connection manager in the share sheets of the device. This means that whenever you click share on the device you can find these shortcuts in the list. At the bottom of this list you can find an option to edit the shown entries. Here you can add the Orange Share shortcuts to your favorites if you want them to always be at the top.

This shortcut is always needed to manage the connections to the host computer running Orange Share. It is the only shortcut that needs to be run directly from the Shortcuts App or added to the home screen and run from there.

The device is able to save multiple connections while only one can be active at a time. In order to do so, some files will be created on your iCloud (/Shortcuts/orangeshare/). To create and choose between available connections, this shortcut offers a few options to choose from.

All shortcuts used to share content use Basic Auth to authenticate the device. Therefore, sharing content without a specifically configured connection is not possible.

Show Current Connection

This option will show the available information about the currently active connection.

New From QR Code

This is the most important option. It allows you to scan a QR code from the devices section to establish a new connection. When a connection is scanned a second time, it will update some settings like the host ip.

You can also use this option to switch the active connection as it will always set the scanned connection as the active one.

New Manual Connection

With this option you can create a new connection by entering all the connection details yourself. The usage of this option is more advanced and not advised if you do not know what you are doing.

Choose from Existing

When choosing this option, you will be prompted with a list of all saved connections. From this list you can choose which connection to activate.

Delete Connections

This option lets you choose from saved connections to delete them.

Forget all Connections

This option will delete all currently saved connections.

Toggle Synced Connection Settings

Connection settings can be synced over iCloud. When this is enabled, all your devices will use the same connection settings. This includes both the active connection and saved connections.

The open shortcut will open content it its respective application on the computer.

For URLs, it will open the website in the default browser, any other files will be opened in their default applications. To make this work, files are saved in a temporary folder.

The save shortcut will open a save-file-dialog on the computer with which shared content can be saved anywhere on the computer.

The clipboard shortcut will copy any text to the clipboard of the computer. Once copied, it can be pasted anywhere.
